streamsx.dps

The Distributed Process Store (DPS) toolkit for Streams allows Streams applications written in SPL, C++ or Java to connect to external Key-Value stores such as Redis.

Version 3.2.1 of the toolkit with support for Streams 4.2 and higher is now available!
